## サービス概要
スマホが替わっても写真を引き継いで保存。気づけばたくさんの写真が、見返すこともなくたまっている方も多いのではないでしょうか。
PhotoUp!(フォタップ)では手軽に個展を開くことができます。
インスタグラムに載せるにはちょっと古い、でも誰かと共有したい過去の写真がある、そんな方におすすめです。

## 主な機能
・ユーザー登録・ログイン・ログアウト機能

・個展の新規作成・編集・削除機能

・個展内の写真投稿機能

・お気に入り登録機能
